#  SatelliteGuard

WireGuard VPN client for iOS, tvOS & macOS. Synced encrypted (E2EE with "Advanced data protection" enabled) using CloudKit. Uses WireGuardKit (`wireguard-go`).

## Working

- CloudKit sync
- NetworkExtension
- TunnelProvider
- iOS, tvOS
